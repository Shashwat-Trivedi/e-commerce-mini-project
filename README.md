# Startup Model Planner for E-Commerce Businesses

A mini-project web app that recommends the best e-commerce startup model based on strategic inputs.

## What it does

- Collects startup goals and constraints through a 10-point planner.
- Uses a weighted logic engine to score four models:
  - Subscription Commerce
  - Affiliate-Led Commerce
  - Direct-to-Consumer (D2C)
  - Two-Sided Marketplace
- Displays ranked fit with confidence level and explainable reasons.
- Suggests a hybrid path when top scores are close.

## Why this is academically relevant

This project converts **E-Business Fundamentals** into a practical decision system:

- Revenue model fit (recurring vs campaign-driven)
- Capital and inventory intensity
- Brand-control strategy
- Retention/LTV orientation
- Platform/network-effect readiness
- Compliance maturity for partner/endorsement-heavy strategies

## Files

- `index.html` → App structure and sections
- `style.css` → Creative/professional UI theme + animation
- `script.js` → Dynamic form generation + recommendation logic

## Run

Open `index.html` in any modern browser.
